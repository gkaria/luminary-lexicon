## Recommended Images

1. **Profile/Portrait** - Main image of the person
2. **Infographics** - Visual summaries of key concepts
3. **Diagrams** - Illustrations of frameworks or models
4. **Historical Photos** - Contextual images
5. **Charts/Graphs** - Data visualizations if relevant

## Image Guidelines

- **Format**: PNG or JPG preferred
- **Size**: Optimize for web (typically < 500KB per image)
- **Naming**: Use descriptive names with hyphens (e.g., `alan-turing-portrait.png`)
- **Alt Text**: Include descriptive alt text in HTML for accessibility
- **Copyright**: Ensure you have rights to use the images
  - Public domain
  - Creative Commons with proper attribution
  - Original creations
  - Fair use (for educational/commentary purposes)
